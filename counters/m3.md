# Apple M3 PMC Events

Source: `/usr/share/kpep/cpu_100000c_2_8765edea.plist`

## Fixed Counters

Always active, not programmable.

| Slot | Name | Alias |
|------|------|-------|
| 0 | FIXED_CYCLES | Cycles |
| 1 | FIXED_INSTRUCTIONS | Instructions |

## Configurable Events (62)

8 counter slots available. Events with a `counters_mask` can only be assigned to specific slots.

### Slot Constraints

- **any slot** -- can be assigned to slots 0-7
- **mask=0xe0** -- restricted to slots 5, 6, 7
- **mask=0x80** -- restricted to slot 7 only

## Notes

- **Speculative vs Non-speculative**: Events ending in `_NONSPEC` count only architecturally retired operations.
- **vs M2**: M3 adds `MAP_UOP`, `SCHEDULE_UOP_ANY`, `MAP_DISPATCH_BUBBLE_IC`, `MAP_DISPATCH_BUBBLE_ITLB`, `LDST_UNIT_OLD_L1D_CACHE_MISS`, `LDST_UNIT_WAITING_OLD_L1D_CACHE_MISS`. Event numbers shifted significantly (e.g. `L1D_CACHE_MISS_LD` 163 -> 1443).
- **vs M4**: M3 lacks ARM_STALL* TopDown events, SME events, `INST_BRANCH_COND`, and additional MAP/scheduler events.
