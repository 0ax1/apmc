# Apple M1 PMC Events

Source: `/usr/share/kpep/cpu_100000c_2_1b588bb3.plist`

## Fixed Counters

Always active, not programmable.

| Slot | Name | Alias |
|------|------|-------|
| 0 | FIXED_CYCLES | Cycles |
| 1 | FIXED_INSTRUCTIONS | Instructions |

## Configurable Events (56)

8 counter slots available. Events with a `counters_mask` can only be assigned to specific slots.

### Slot Constraints

- **any slot** -- can be assigned to slots 0-7
- **mask=0xe0** -- restricted to slots 5, 6, 7
- **mask=0x80** -- restricted to slot 7 only

## Notes

- **Speculative vs Non-speculative**: Events ending in `_NONSPEC` count only architecturally retired operations.
- **vs M2**: M1 lacks `INST_SIMD_ALU_VEC`. M1 has `SCHEDULE_UOP` (replaced by `SCHEDULE_UOP_ANY` on M3+).
