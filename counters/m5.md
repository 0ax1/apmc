# Apple M5 PMC Events

Source: `/usr/share/kpep/cpu_100000c_2_da9d04c3.plist`

The M5 has the same PMC event set as the [M4](m4.md). See that document for the full event listing.

## Summary

- **Fixed counters**: 2 (FIXED_CYCLES, FIXED_INSTRUCTIONS)
- **Configurable events**: 101
- **Configurable counter slots**: 8 (kernel reserves 3 until `force_all_ctrs_set(1)`)
- **Slot masks**: 0xfc (slots 2-7), 0x80 (slot 7 only)
- **New vs M3**: ARM_STALL* TopDown events, SME engine events, INST_BRANCH_COND, additional MAP/scheduler events
